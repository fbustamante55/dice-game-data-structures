# Dice Game – Data Structures

A console-based dice game implemented in Java, demonstrating the use of data structures to manage game state, players, and scoring.

## Technologies
- Java 11+
- Data Structures

## How to Play
```bash
javac Main.java
java Main
```
Follow the on-screen prompts to roll dice and compete against other players.

## Features
- Multi-player support
- Score tracking with data structures
- Turn-based gameplay
- Win condition detection
